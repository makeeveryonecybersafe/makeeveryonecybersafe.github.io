// Find Files by Name
find . -name "*.log"

Locates all log files in current directory and subdirectories.

// Check Listening Ports
netstat -tuln

Displays TCP/UDP ports currently listening for connections.

// File Permissions (Secure)
chmod 600 config.yaml

Sets file to be readable/writable only by the owner.

// SSH Key Gen
ssh-keygen -t ed25519 -C "email"

Generates a secure Ed25519 SSH key pair.
